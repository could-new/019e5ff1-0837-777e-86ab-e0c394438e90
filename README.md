# Methodology Flowchart

This Flutter application provides a neat, professional, and editable methodology flowchart designed for a final project report. It features a pure white background, rounded rectangular boxes, and clear arrow connectors arranged in a clean sequential flow from top to bottom.

## Features

*   **Interactive Flowchart**: Panning and zooming enabled via `InteractiveViewer`.
*   **Professional Academic Appearance**: Minimalistic and clean design suitable for thesis and project documentation.
*   **Responsive Layout**: Built with a CustomPainter-based layout to accurately position steps and draw connections dynamically.
*   **Sequential Flow**: Details the experimental process, from the selection of raw materials to the addition of encapsulated beads and fermentation.

## Flowchart Steps Included

*   Selection of Raw Materials
*   Washing, Slicing, Soaking, Draining, Rinsing, Grinding
*   Vitamin C Extraction (with Variations 1-3)
*   Extraction and Selection of Best Yield
*   Quantification and Encapsulation (Pectin + Orange Peel Solution)
*   Hardening, Washing, and Storage
*   Juice Preparation (Bilimbi Juice + Mint Juice + Monk Sugar)
*   Storage and Fermentation

## Tech Stack

*   Flutter / Dart
*   CustomPainter for drawing relational arrows.

## Setup & Run

1.  Ensure you have Flutter installed.
2.  Run `flutter pub get`.
3.  Run `flutter run` to launch the application.
